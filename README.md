# 🧬 AI-Powered Biomedical Health Analytics Dashboard

A professional biomedical data analytics dashboard that monitors patient vital signs, predicts disease risk, and visualizes health trends using interactive charts.

This project demonstrates biomedical data analysis, visualization, and AI-based risk prediction using modern web technologies.

# 📊 Project Overview

The **Biomedical Health Analytics Dashboard** simulates a real-world patient monitoring system used in hospitals and biomedical research environments.

The system allows users to:

* Enter patient vital signs
* Predict health risk using AI-based logic
* Analyze biomedical trends through visual charts
* Generate automated PDF reports
* Store patient data locally for continuous monitoring

This project integrates **biomedical engineering concepts with data analytics and visualization**.

# 🧠 Key Features

### 🩺 Patient Data Monitoring

Users can input patient biomedical data including:

* Age
* Heart Rate
* Oxygen Saturation
* Body Temperature

### 🤖 AI-Based Risk Prediction

The system predicts patient risk levels:

* Low Risk
* Medium Risk
* High Risk

Prediction is based on biomedical thresholds commonly used in clinical monitoring.

### 📈 Interactive Data Visualization

The dashboard displays biomedical data using interactive charts such as:

* Heart Rate Trend Analysis
* Oxygen Saturation Monitoring
* Patient Risk Distribution

Charts are dynamically updated whenever new patient data is entered.

### 📄 Automated PDF Report Generation

The system can generate a **downloadable biomedical analysis report** summarizing:

* Total patients analyzed
* Key health insights
* Risk observations

### 💾 Data Storage

Patient data is stored using browser **Local Storage**, allowing the dashboard to retain previously entered data.

# 🧬 Biomedical Insights

The system provides analytical insights such as:

1. Patients with **oxygen saturation below 94% show increased infection risk**.

2. **Body temperature above 38°C** may indicate possible infection or inflammatory response.

3. Elevated **heart rate above 110 bpm** may indicate cardiovascular stress.

These indicators are commonly monitored in ICU and infectious disease environments.
